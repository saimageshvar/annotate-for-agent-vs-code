# Categories carry intent

| Category | Meaning |
|----------|---------|
| **Bug** | Something is not working as intended |
| **Refactor** | Code works but structure could improve |
| **Nit** | Small preference, optional to address |
| **Question** | Asking for clarification, not a change |
| **Praise** | Positive note on good work |
| **Suggestion** | An alternative to consider |

Each category color-tags the gutter icon and tag pill. On export, a category legend can be included so the agent reads the intent the same way you wrote it.

## Picking fast

- **Panel:** click pill, or `Alt+1`–`6` from anywhere in the form
- **Quick:** press number key in the QuickPick
