# Create an annotation

1. Select a line or range in the editor
2. Press `Ctrl`/`Cmd+Shift+A` — runs your default mode
3. Or press `Ctrl`/`Cmd+Shift+Q` — always Quick mode
4. Right-click also works: *Annotate for Agent: Annotate*

The annotation captures the file path, range, snippet, and category. It appears in the **Annotations** activity bar pane.
